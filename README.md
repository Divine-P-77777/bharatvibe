# 🇮🇳 BharatVibes

**A cultural journey through India — powered by the people, for the world.**  
🌸 Built with ❤️ by `DesiDynamiX`

---

## 📌 Overview

**BharatVibes** is a modern full-stack Progressive Web App (PWA) built with **Next.js (TypeScript)**, **Supabase**, and **Cloudinary** to showcase the incredible cultural diversity of India. Users can explore and contribute to a community-driven collection of locations, foods, traditions, and blogs — all backed by a media-rich experience using images and videos.

---

## ✨ Features

- 🧭 **Explore India's Culture** – Locations, cuisines, clothing, festivals & traditions
- 📸 **Media Uploads** – Users can share **images/videos** using **Cloudinary**
- 🗣️ **Community Engagement** – Comment, vote, and share cultural content
- 🔐 **Google Auth** – Seamless login via **NextAuth with Supabase OAuth**
- 🎖️ **Reward System** – Earn coins via engagement, convert to real money via Razorpay
- 💬 **Post Reactions & Comments** – Like/dislike and interact in real-time
- 🌍 **Foreign Tourist Friendly** – Insightful for people planning to visit India
- 🛡️ **Content Moderation** – Block/report inappropriate content
- 🌈 **Dynamic UI/UX** – 3D parallax cards, smooth animations with Framer Motion
